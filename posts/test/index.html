1. 无重复字符的最长字串

题号：3

难度：中等

题目链接

给定一个字符串，请你找出其中不含有重复字符的 最长子串 的长度。

示例：

输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。

题解

解1：滑动窗口

    public int lengthOfLongestSubstring(String s) {
        // 哈希集合，记录每个字符是否出现过
        Set<Character> occ = new HashSet<Character>();
        int n = s.length();
        // 右指针，初始值为 -1，相当于我们在字符串的左边界的左侧，还没有开始移动
        int rk = -1, ans = 0;
        for (int i = 0; i < n; ++i) {
            if (i != 0) {
                // 左指针向右移动一格，移除一个字符
                occ.remove(s.charAt(i - 1));
            }
            while (rk + 1 < n && !occ.contains(s.charAt(rk + 1))) {
                // 不断地移动右指针
                occ.add(s.charAt(rk + 1));
                ++rk;
            }
            // 第 i 到 rk 个字符是一个极长的无重复字符子串
            ans = Math.max(ans, rk - i + 1);
        }
        return ans;
    }

解2：一次遍历

    public int lengthOfLongestSubstring(String s) {
        char[] chars = s.toCharArray();
        //map记录每个元素在遍历过程中最新出现的位置
        Map<Character, Integer> map = new HashMap<>();

        int nowSum = 0, res = 0;

        for (int i = 0; i < chars.length; i++) {
            //如果之前没有出现过chars[i]
            if (!map.containsKey(chars[i])) {
                nowSum++;
            } else {
                //如果在之前出现，则判断是在当前进行扩展的子串之前还是子串之中
                if ((i - map.get(chars[i])) <= nowSum) {
                    //在子串之中出现的重复，那么从之前那个元素开始计算当前子串的长度
                    nowSum = (i - map.get(chars[i]));
                } else {
                    //如果实在当前子串之前出现的重复，那么不影响当前子串
                    nowSum++;
                }
            }
            //更新元素最新出现位置的下标
            map.put(chars[i], i);

            res = Math.max(res, nowSum);
        }
        return res;
    }

2. 字母异位词分组

题号：49

难度：中等

题目链接

示例：

输入: ["eat", "tea", "tan", "ate", "nat", "bat"]
输出:
[
  ["ate","eat","tea"],
  ["nat","tan"],
  ["bat"]
]

题解

解1：排序

由于互为字母异位词的两个字符串包含的字母相同，因此对两个字符串分别进行排序之后得到的字符串一定是相同的，故可以将排序之后的字符串作为哈希表的键。

    public static List<List<String>> groupAnagrams(String[] strs) {
        //key:排序后的字符数组，value：异位词列表
        Map<String, List<String>> map = new HashMap<String, List<String>>();

        for (int i = 0; i < strs.length; i++) {
            char[] array = strs[i].toCharArray();
            Arrays.sort(array);
            String key = new String(array);

            List<String> list = map.getOrDefault(key, new ArrayList<>());
            list.add(strs[i]);
            map.put(key, list);
        }
        return new ArrayList<List<String>>(map.values());
    }

解2：计数

对每个字符串中每个字符出现的次数进行计数,并按照字母表顺序拼接成字符串，作为map的key

    public static List<List<String>> groupAnagrams(String[] strs) {
        Map<String, List<String>> map = new HashMap<String, List<String>>();
        for (String str : strs) {
            int[] counts = new int[26];
            int length = str.length();
            for (int i = 0; i < length; i++) {
                counts[str.charAt(i) - 'a']++;
            }
            // 将每个出现次数大于 0 的字母和出现次数按顺序拼接成字符串，作为哈希表的键
            StringBuffer sb = new StringBuffer();
            for (int i = 0; i < 26; i++) {
                if (counts[i] != 0) {
                    sb.append((char) ('a' + i));
                    sb.append(counts[i]);
                }
            }
            String key = sb.toString();
            List<String> list = map.getOrDefault(key, new ArrayList<String>());
            list.add(str);
            map.put(key, list);
        }
        System.out.println(map);
        return new ArrayList<List<String>>(map.values());
    }

输入为["abc","bac","qwer","werq"],构建的map和返回结果如下：

{e1q1r1w1=[qwer, werq], a1b1c1=[abc, bac]}
[[qwer, werq], [abc, bac]]

3. 找到字符串中所有字母异位词

题号：438

难度：中等

题目链接

给定一个字符串 s 和一个非空字符串 p，找到 s 中所有是 p 的字母异位词的子串，返回这些子串的起始索引。

字符串只包含小写英文字母，并且字符串 s 和 p 的长度都不超过 20100。

说明：

  • 字母异位词指字母相同，但排列不同的字符串。
  • 不考虑答案输出的顺序。

示例：

输入:
s: "cbaebabacd" p: "abc"

输出:
[0, 6]

解释:
起始索引等于 0 的子串是 "cba", 它是 "abc" 的字母异位词。
起始索引等于 6 的子串是 "bac", 它是 "abc" 的字母异位词。

题解

滑动窗口

    public static List<Integer> findAnagrams(String s, String p) {
        int sLength = s.length();
        int pLength = p.length();
        List<Integer> resList = new ArrayList<>();
        if (sLength < pLength) return resList;

        //记录每个字母出现的次数
        int[] pCount = new int[26];
        int[] sCount = new int[26];
        
        for (int i = 0; i < pLength; i++) {
            ++pCount[p.charAt(i) - 'a'];
            ++sCount[s.charAt(i) - 'a'];
        }

        System.out.println();
        if (Arrays.equals(pCount, sCount)) {
            resList.add(0);
        }

        //每次滑动窗口移动一位，新增加一个元素，移除一个元素
        for (int i = pLength; i < sLength; i++) {
            ++sCount[s.charAt(i) - 'a'];
            --sCount[s.charAt(i - pLength) - 'a'];
            if (Arrays.equals(pCount, sCount)) {
                resList.add(i - pLength + 1);
            }
        }
        return resList;
    }
